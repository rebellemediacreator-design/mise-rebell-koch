AZUBI PRO · KOCH/KÖCHIN (3 Lehrjahre)
=================================

Upload dieses Ordners zu Netlify (Drag & Drop).

Erste Schritte:
1) Seite öffnen → Pro-Azubi Setup erscheint
2) Ausbildungsstart setzen + Name/Betrieb eintragen
3) Start-Screen: täglich/wöchentlich Notizen schreiben
4) "Berichtsheft-PDF" → Druckdialog → Als PDF speichern

Hinweis:
- Export ist "berichtsheft-nah" (Kopfzeile, Zeitraum, Unterschriftenfelder).
- Ob eine Kammer/der Betrieb eine bestimmte Vorlage verlangt, kann regional variieren.

Dateien:
- index.html, styles.css, app.js, content.js
- glossary.js (Glossar PRO)
- datenschutz.html, impressum.html, agb.html
